# Photo Gallery

A fast, responsive photo gallery built with HTML & CSS.

## Setup

1. **Convert images**: Convert your HEIC photos to JPG format (smaller, better browser support)
2. **Add JPGs**: Place `img_1.jpg`, `img_2.jpg`, etc. in this folder
3. **Commit & Deploy**

## Deploy Fast

### Option 1: GitHub Pages (Free, 30 seconds)
```bash
# In your repo settings, enable GitHub Pages from main branch
# Your site will be live at: https://EpKuz6121.github.io/Testing1
```

### Option 2: Netlify (Free, 1 minute)
- Drag & drop this folder to [Netlify](https://netlify.com)
- Instant live URL

### Option 3: Vercel (Free, 1 minute)
```bash
npm i -g vercel
vercel
```

All options give you a live URL instantly!
